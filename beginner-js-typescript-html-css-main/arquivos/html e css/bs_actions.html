Alertas Animados

As classes .fade and .in adicionam um efeito quando a caixa é fechada

× Success! Esta caixa de alerta pode indicar uma ação que ocorreu corretamente.
× Info! Esta caixa de alerta pode indicar um alerta.
× Warning! Esta caixa de alerta pode indicar que a ação precisa de atenção.
× Danger! Esta caixa de alerta pode indicar perigo ou erro.
